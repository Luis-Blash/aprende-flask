Flask

¿Qué necesitas para empezar?

Python

Tener instalado la version de python3 puedes descargarlo desde:

python

PIP

Al instalar python vendra por defecto pip, el cual ya no sera necesario instalarlo, en caso de tener una version inferior de pip, puedes copiar el siguiente comando:

Ubuntu
sudo apt-get install python3-pip

virtualenv

Es recomendable que trabajes utilizando virtualenv para crear entornos aislados para Python, en los que es posible instalar paquetes sin interferir con otros virtualenvs ni con los paquetes de Python del sistema.
puedes instalarlo con el siguiente comando

PIP
pip install virtualenv

Trabaja en docker

Tambien puedes trabajar en docker de esta forma no necesitaras instalar nada en tu computadora y podras hacer tus proyectos de forma sencilla, te recomiendo que veas el video de Flask en docker.
